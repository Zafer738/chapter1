HTML & CSS Quick-Reference Guide

A helpful guide covering essential HTML tags and CSS properties with examples.

Common HTML Tags

  • <h1> - <h6>: Heading for the content.
  • <p>: Defines a paragraph.
  • <ul> & <ol>: Unordered and ordered lists.
  • <li>: List item used inside lists.
  • <a href="">: Creates hyperlinks.
  • <img src="" alt="">: Displays images.
  • <div>: Defines a section in a document.
  • <span>: Inline container for styling.
  • <table>: Creates tables.
  • <form>: Creates input forms.

Common CSS Properties

  • color: Sets text color.
  • background-color: Sets background color.
  • font-family: Defines font style.
  • margin: Controls spacing outside elements.
  • padding: Controls spacing inside elements.
  • border: Adds borders around elements.
  • text-align: Aligns text (left, center, right).
  • box-shadow: Adds shadow effects.

Example Code


        <h1>Hello, Sir!</h1>
        <p>This is a paragraph.</p>

        p {
            color: blue;
            font-size: 16px;
        }
